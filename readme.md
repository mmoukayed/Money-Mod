# 💰 Money Mod

**Money Mod** adds a full banking and currency system to Minecraft, bringing realistic financial mechanics into your world. Use physical cash, manage balances with credit cards, withdraw money from ATMs, and find naturally generated banks across the world.

Built for **NeoForge 1.21.8**, this mod introduces a new way to handle economy systems for survival worlds and multiplayer servers.

---

# ✨ Features

## 💵 Physical Currency System
- Multiple **cash bills and coins**
- Physical items that represent real currency
- Designed for trading and server economies

## 💳 Credit Card System
- Credit cards store a **digital balance**
- Protected by **PIN codes**
- Cards can be upgraded to a **Gold Credit Card**

## 🏧 ATMs
- Deposit and withdraw money
- Interact with your credit card balance
- Provides easy access to your funds

## 🧾 Payment Terminals
- Used for transactions between players
- Accept credit card payments

## 💎 Emerald Exchange
- Convert **emeralds into cash**
- Allows integration with vanilla trading systems

## 🏦 Naturally Generating Banks
Banks generate naturally in the world and contain:
- ATMs
- Payment terminals
- Banking blocks and decorations

Banks can spawn **anywhere in the world** and act as economic hubs.

# 📦 Installation

1. Install **NeoForge** for **Minecraft 1.21.8**
2. Download the latest **Money Mod `.jar` file** from the **Releases** tab
3. Place the file inside your:


.minecraft/mods


4. Launch the game

No additional dependencies are required.

---

# ⌨ Commands

### Credit Card Commands


`credit add <player> <amount>`

Adds the specified amount to the **credit card the player is holding**.


`credit subtract <player> <amount>`

Removes the specified amount from the **credit card the player is holding**.


`credit balance`

Displays the **balance of the credit card you are currently holding**.

# 🌐 Multiplayer Support
